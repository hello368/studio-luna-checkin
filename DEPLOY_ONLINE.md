## Phase 1: Database Setup (Supabase) ðŸ—„ï¸
1.  Go to **[supabase.com](https://supabase.com)** and Sign Up / Log In.
2.  Click **"New Project"**.
3.  Name it: `Studio Luna Checkin`.
4.  Set a strong database password.
5.  Region: Choose one close to you (e.g., US West).
6.  Click **"Create new project"**.
    *   *Wait about 2 minutes for it to set up.*

### Create the Table
1.  Once the project is ready, click on **"SQL Editor"** (icon looks like a terminal `>_` on the left sidebar).
2.  Click **"New Query"**.
3.  Paste this code into the box:
    ```sql
    create table public.visits (
      id bigint generated by default as identity primary key,
      created_at timestamp with time zone default timezone('utc'::text, now()) not null,
      name text,
      phone text,
      email text,
      treatment text,
      consent_text text,
      signature text
    );
    ```
4.  Click **"RUN"** (green button).
    *   You should see "Success" in the results.

### Get Your API Keys (Save these for Phase 2!)
1.  Click on **"Project Settings"** (gear icon âš™ï¸ at the very bottom left).
2.  Click on **"API"**.
3.  Look for **"Project URL"**. -> Copy this! (This is your `SUPABASE_URL`)
4.  Look for **"anon" / "public"** key. -> Copy this! (This is your `SUPABASE_KEY`)

## Phase 2: Deploy to Vercel â˜ï¸
1.  Go to **[vercel.com](https://vercel.com)** and Log In.
2.  Click **"Add New..."** -> **"Project"**.
3.  Next to `studio-luna-checkin` (from GitHub), click **"Import"**.
4.  **Configure Project**:
    *   **Framework Preset**: Leave as "Other".
    *   **Root Directory**: Leave as `./`.
5.  **Environment Variables** (Crucial Step!):
    *   Click to expand the "Environment Variables" section.
    *   Add Variable 1:
        *   Name: `SUPABASE_URL`
        *   Value: *(Paste the URL from Phase 1)*
    *   Add Variable 2:
        *   Name: `SUPABASE_KEY`
        *   Value: *(Paste the Key from Phase 1)*
6.  Click **"Deploy"**.

## Phase 3: You're Live! ðŸŽ‰
1.  Wait about 1 minute.
2.  Vercel will give you a domain (e.g., `studio-luna-checkin.vercel.app`).
3.  Click it to visit your live app!
4.  Test it:
    *   Fill out the form.
    *   Sign it.
    *   Check that it says "Success".
    *   Visit `/admin.html` to see the entry!

---
## Troubleshooting
*   **"Internal Server Error"**: Usually means Environment Variables are wrong. Go to Vercel Settings > Environment Variables and double check them.
*   **Form submits but dashboard empty**: Check if you ran the SQL query in Supabase correctly.
